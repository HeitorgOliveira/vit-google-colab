# ViT Usando o Google Colab

Esse repositório destina-se a ajudar quem busca importar o modelo de ViT no google colab para realizar o ajuste fino e avaliá-lo utilizando
um dataset próprio, de preferência armazenado no drive.

Como esse repositório foi feito para ser utilizado idealmente no google colab, não é necessário baixar dependências, mas caso insista em utilizar
este código em outro ambiente, baixe as dependências utilizando 

``` pip install requirements.txt ```
